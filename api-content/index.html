初一的时候，因为大家不是很熟，所以没什么特别，只是努力想学好，可考试成绩一下来，我真的不敢相信，自己居然排倒数。不过，却多了几个“朋友”。我曾经试着向成绩好的同学接近，可他们似乎认定了中国的一句古话“道不同，不相为谋”。每次我去请教，他们的态度都会变的好不耐烦。于是，在一气之下决定放弃。踏出了走向“堕落”的第一步。 初二，我由一个中下生变成了一个名副其实的差生，从此“不学无术”。下学期，通过“好友”认识了在社会上的混混，从此和他们混在一起，荒废学业。不过，那时候也好开心，因为没有人管我们。每天中午和放学，他们都会来，这让我们感到很“自豪”。不过，也有东窗事发的时候，有一，两次，我和他们在一起被老师发现，于是被告之家长。但这似乎也没阻断我和他们的交往…… 初三上学期，冲刺的学年。这学期，我彻底的和他们断绝了来往，准备扑在学习上。没想到几天下来现实证明了我不行，因为每次往深了想问题的时候，都会头痛，于是再一次的放弃。 结束回忆，面壁反醒。知道自己不能活在回忆里。于是，准备放手一搏，试着在这一学期闯出属于自己的明天。
